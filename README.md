1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser** and navigate to the URL shown in the terminal (typically `http://localhost:5173`)

## How It Works

1. Select a customer from the dropdown
2. Click "Optimize (Top 10)" to filter the best companies
3. Click "AI Optimize (Client History)" to see AI-powered confidence scores
4. View color-coded results based on confidence levels
